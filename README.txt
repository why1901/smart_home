此次项目是模仿一个简易版的智能家居项目，其中基于ARM Cortex-A53 S5P6818发板设计与实现。

其中主要实现了手机、服务器、开发板基于tcp协议的三者联合，可通过手机连接服务器给开发板终端发送指令进行开灯/关灯功能以及蜂鸣器的开关功能。
同时在开发板终端可以操控开关灯以及蜂鸣器，另外在操作的同时，可以将其操作的信息通过服务器发送给连接服务器的手机上。
简易地实现了如今智能家居的大致过程。


在此次的项目中主要设计的部分是服务端和开发板终端，手机终端采用的是“有人网络助手”连接服务器。


在服务器端，主要采用IO多路利用的方式监听socket，在服务器代码中使用内核链表来记录手机终端连接时的fd，而开发板终端则会被单独记录出来。
当开发板数据传输过来时，服务器根据链表中的fd转发所接收的数据；当手机终端发送来数据，则转发到开发板上。



在开发板终端采用线程的方式分别用来监听连接服务器是否有数据传输来，另外线程则实现监听lcd屏幕
